入门到放弃9

DOM树形文档操作

  • offsetheight 元素自身的高度
  • scrollheight 内部内容的高度
  • document 没有html这个属性 如果要找html标签就用documentElement属性

  • window.getComputedStyle 能够获取到所有最终生效的样式

    • IE678有问题 支持currentStyle 属性

  • math.cilen 天花板

  • math.floor 地板

  • math.round 就近取整0.5以上或者0.5都上取

  • clientheight 宽度+内边距

  • 三大系列总结 都是dom的

    • offset scroll cilen

  • onresize属性可以用来获取或设置当前窗口的resize事件的事件处理函数是window的方法

  • event.clientX 窗口
  • event.pageX 页面
  • event.screeX 屏幕
    • 最常用的鼠标在页面中的位置 但是有兼容问题

事件

  • event||window.event;
  • var pagX = event.pageX||event.clientX+document.documentElement.scrollLeft;

  • var pagY = event.pageY||event.clientY+document.documentElement.scrollTop;

  • addEventListener 的三个值 事件监听器 正常浏览器

    • type事件类型 字符串比如”click”
    • listener事件处理函数 function(){}
    • 是否使用捕获 先写false

  • removeEventListener 移除事件监听器 正常浏览器

    • 三个参数 事件 回调函数 是否捕获
    • 第一个还是”click”
    • 第二个是 传递有名字的函数方法 匿名的会找不到
    • 第三个还是 false

  • attachEvent 事件监听器 IE678注册

    • type事件类型 字符串比如”click”要加on　“onclick”
    • listener事件处理函数 function(){}
    • 是否使用捕获 先写false

  • detachEvent 解绑事件 IE678解绑

    • 第一个还是”click”
    • 第二个是 传递有名字的函数方法 匿名的会找不到
    • 第三个还是 false

  • 冒泡是里面到外面 捕获是从外面到最里面

模块化

  • 模块化开发说的简单些就是把一堆方法函数放进去一个对象
    -然后通过对象管理函数方法 每次调用这些方法的时候如果有形参那你一定要记得传参

  • e.keyCode 获取当前按键的数字

正则表达式

  • 数字返回要求true
  • 字母不符合要求就返回false

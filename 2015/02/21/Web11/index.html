入门到放弃11

jQuery

API

  • jquery 中只传入属性名，代表获取属性值
  • jquery 中传入对象 可以同时设置多个属性
  • jquery 中传入属性名和属性值就是代表设置属性
  • jquery 获取到的样式直接就是计算完成之后的样式
  • jquery 中对象的方式设置样式属性名可以不加字符串但是属性值要写

  • jquery 在设置样式的时候传入字符串可以是font-size 也可以是fontSize

    • val() 获取文本值
    • text() 获取文本内容
    • empty() 清空当前元素
    • remove() 删除当前元素
    • html() 不传递参数代表获取标签结构代码
    • attr() 获取src属性 title替换提示文本 alt替换文本
    • removeattr 这样删除比较干净

  • toggleClass 切换类名有就切换 没有就添加

  • addClass 添加指定类名 可以同时添加多个 空格分开
  • hasClass 判断是否有指定类名 有就是true 没有返回false

  • removeClass 删除指定类名 不传移除的类名就移除所有类名

  • jqeury show 方法可以传递三个值 fase normal slow

    • 第一个是速度 第二个是回调函数

  • jquery hide 方法如果传入的是不认识的字符串就按照normal执行

    • 第一个是速度 第二个是回调函数

  • slideDown 划入事件 不传参数都会有动画

  • slideUp 划出事件 如同sideDown完全一样

  • show hide 改变的是高度 宽度 以及透明度

  • slideDown slideUp 改变的是高度

  • fadein 淡入的事件和划入完全一样

  • fadeout 淡出的事件和划出完全一样
  • fadetoggle 淡入淡出的切换
  • fadeto 淡淡的达到 1.速度 2.目标值 3.效果动画 4.回调函数

自定义动画

  • animate 1.属性 2.速度 3.动画效果 4.回调函数
    • 自定义动画只能改变数值类的属性 单一属性 合写可能会出现问题 列如背景色字体无法改变
    • swing 速度是 满快慢 linear 匀速动画 如果不传easing这个参数 默认就是swing
    • 调用动画之前先停止动画 然后就不会存在都离开了 还有好多动画还在队列未执行的效果
    • $(“li”).stop().animate({width:”240px”},3000,”swing”)

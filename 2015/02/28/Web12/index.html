入门到放弃12

jQuery

jq中专门宽度和高度以及设置的方法 把width改成height就是高度的方法 传参就是设置

  • .innerwidth()实际宽度和内边距
  • .outerwindth()宽度和内边距以及边框线
  • .width()专门获取元素的宽度 得到的是数组 可直接运算

  • .outerwindth(true)传递一个true 得到的是内边距+边框线+外边距+宽度

    • js中offsetwidth clien scrolltop 都是只能获取 不能设置数值
    • jq中//获取页面滚动距离上面的高度$(document).scrollTop();可设置数值

  • 委托事件 1事件 2选择器 3回调函数

  • 委托事件 一定要委托给页面上已经存在的元素

  • jquery中

    • e.stopPropagation();//阻止冒泡 （没有兼容问题）
    • //e.preventDefault();//阻止默认行为 （但是不阻止冒泡）

    • return false;//即阻止默认行为 又阻止冒泡

    • //target 触发事件的最根源的元素

    • //currentTarget 当前事件所属的元素（相当于this）

    • //delegateTarget 当前事件被委托的元素（委托在谁身上了）

    • console.log(e.currentTarget);//button

    • console.log(e.target);//button
    • console.log(e.delegateTarget);//div 委托给谁了就是谁

    看插件的test就知道要怎么用插件了

入门到放弃15

C3

背景

  • background-size 背景尺寸

    • cover 完全铺满 contain 完全显示

  • background-origin 背景原点 从那里开始平铺

    • border-box
    • padding-box
    • content-box

  • background-clip 裁剪

    • border-box
    • padding-box

    • content-box

    • 多背景 逗号

渐变

  • linear-gradient 线性渐变

    • 方向
    • 起始颜色
    • 结束颜色

  • radial-gradient

    • 半径
    • 原点
    • 起始颜色
    • 结束颜色

过渡

  • 1.所有的属性状态发生改变都可以产生过渡
  • 2.但是有些属性很特殊 display:none display:block 看不到
  • 3.all 所有的属性都可以产生过渡效果
  • 4.可以指定某一个属性 或者 多个属性 产生过渡

2d转换

  • transform 转换属性

    • transform 转换属性
    • scale(arg1,arg2) 比例的缩放
    • arg1 x轴方向的缩放
    • arg2 y轴方向的缩放
    • 1 原始比例
    • 比1大 放大
    • 比1小 缩小

  • scale 缩放

    • transform 转换属性
    • scale(arg1,arg2) 比例的缩放
    • arg1 x轴方向的缩放
    • arg2 y轴方向的缩放
    • 1 原始比例
    • 比1大 放大
    • 比1小 缩小

  • translate 位移

    • transform 转换属性
    • translate(arg1,arg2) 元素最终位置的位移
    • arg1 x轴方向移动
    • arg2 y轴方向移动
      • 1.位移和定位没有关系 位移是根据元素的最终状态进行位移

  • rotate 旋转

    • transform 转换属性
    • rotate(arg1) 旋转
    • arg1 旋转的角度 如果是顺时针旋转 角度是正的 反之 逆时针旋转就是负的角度

  • skew 倾斜

    • transform 转换属性
    • skew(arg1,arg2) 元素倾斜
    • arg1 x轴方向倾斜
    • arg2 y轴方向倾斜

    • 尽可能使用一个方向的倾斜

    • 如果是X轴方向的倾斜

    • 如果是正角度的倾斜 逆时针的倾斜

    • 如果是负角度的倾斜 顺时针的倾斜

    • 如果是Y轴方向的倾斜

    • 如果是负角度的倾斜 逆时针的倾斜
    • 如果是正角度的倾斜 顺时针的倾斜

  • transform-origin 转换原点

    • 转换 原点 设置
    • top left right bottom center确定旋转中心
    • px 定位的方式

3d转换

  • css 3d 坐标轴

  • perspective 透视 参数 设置 视距

    • 让3d转换的效果 可以呈现出近大远小的效果
    • /*透视
    • 参数 视距
    • */
    • perspective: 100px;

  • transform-style: preserve-3d; 透视 3D效果成呈现

  • rotateX

    • rotateX 绕着X轴转
    • 顺子轴的正方向看
    • 如果是逆时针 角度是正值
    • 如果是顺时针 角度是负值

  • rotateY

  • rotateZ

  • translateX

  • translateY
  • translateZ

3d转换

  • transform-style flat 2d呈现 preserve-3d 3d呈现

  • 顺着轴的正方向看 逆时针 正角度 顺时针 负角度 立方体

  • 一张背景图在原地翻转的时候得写

    - transform:translateZ(2px);谷歌是2PX 火狐是1PX才会有旋转效果

动画

  • animation

      1. @keyframes { } 定义动画
      1. from to 开始 结束

    • animation-name 动画序列名称

    • animation-duration 时间
    • animation-timing-function 速度 steps(步数)
    • animation-delay 延时
    • animation-fill-mode 动画结束后状态 forwards backwards
    • animation-iteration-count 执行次数
    • animation-direction 是否逆播放 alternate
    • animation-play-state 播放状态 running paused

伸缩布局

  • display:flex 定义一个伸缩容器
  • flex-direction: column; 改变主轴方向 row
  • flex-wrap:nowrap; wrap 换行

  • flex:1 分配剩余的宽度 当前容器占多少份

  • 1、伸缩布局

    • 标准流
      • 行内没有宽高 块级 独占一行 行内块 一行显示多个 有宽高

    • 浮动 ： 左浮动 有浮动

    • 定位 ： 基准点 边偏移 top left bottom right right:20px;

    • 伸缩布局：
      • 主轴 侧轴
      • 默认主轴从左向右，侧轴默认从上到下
      • 子盒子的排列方向 是按照主轴的方向
      • dispaly:flex:
    • 主轴反向：
      • flex-direction:
      • row row-reverse column column-reverse

    • 主轴的对齐方式：

      • justify-content:
      • flex-start:从主轴开始的地方对齐
      • flex-end:从主轴结束的地方对齐
      • center :居中
      • space-aruond:平分父盒空间
      • space-between: 两边对齐

    • 侧轴对齐方式：

      • align-items:
        • flex-start:从侧轴开始的地方对齐
        • flex-end:从侧轴结束的地方对齐
        • center:居中
        • stretch:侧轴拉伸 最大化的充满侧轴方向(不要手动设置高度)
      • 是否换行：
        • flex-wrap: nowrap(默认) wrap

    • 伸缩比例：

      • flex:
      • 直接给子盒子设置伸缩臂比例 ：所占有的份数 （相对于父盒子的宽度）
      • 不参与划分的子盒子 ，不用设置flex即可 ，宽高默认不变

    • 字体图标：

      - 声明字体：告诉浏览器去哪找字体
- 在伪元素中使用：
  * &#xe601;
  * 注意写法：content:"\e601";
    • 定义申明字体

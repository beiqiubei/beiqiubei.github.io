注意点

  • 字符串相加会拼接，数字相加会等于结果
  • 比如“11”+“11” 结果1111 1+1 结果2
  • 什么类型的数据+上字符串都会默认隐式转换成字符串
  • number数字类型 var num=18 可以直接看出当前类型的字面量18表示字面量为2
  • 千万不要以两个浮点数做为判断依据（判断范围可以）

数字的数字范围

  • 最大值 max_value infinity 正的无穷大
  • 最小值 min_value -infinity 负的无穷大

数值检测

  • NaN 的数据类型是 number
  • NaN 自己和自己都不相等
  • isNan 能检测一个数是否是数字 不是数字就返回 true 是数字就返回flase

字符串类型

  • 字符串用双引号还是单引号都可以 但是不要混在一起用

  • 将来在操作html代码的时候建议先单引号在双引号

    • 字符串的不可变性 指的是内存中的不可变性
    • 千万别大量拼接字符串，因为字符串的不可变性，导致大量拼接字符串会非常消耗资源

    • 字符串和数字相加的时候 +数字相加 字符相连 只要有字符就会吧其他类型的值转化成
      字符串进行拼接

    • 逗逗加加 如何吧一个写死的变量如何吧数据改成活的

    • var minite = 39;
    • var str = “现在时间是” + minite + “分”;

boolean布尔型

  • Booleam()转换字符串中的”” 数值中的0和nan null undefined 转换都是flase 其他都是true
  • 隐式转化 比如boolean=！！num 就可以成功转换

undefined

  • undefined是特殊类型 说明变量但未赋值

转化string三种方式

  • Tostring()
  • undefined和null不能调用tostring()方法
  • tostring()什么类型都可以转化成字符串 通用的方式
  • 隐式转化 定义的变量+上一个空字符串就可以转化成字符串 将来最常用的方式

转化数字

  • Number() 转化成数字的方法 有一个特点转换的东西中只要有不认识的就是nan
  • null和flase转化是0 undefined转化会是nan ture会是1
  • 只能转换纯数字的的字符串，如果包含了其他字母返回nan
  • parseint和parsefloat 是专门吧字符串转换成数字 把认识的数字取出来 不认识的部分忽略
  • parseint 不认小数 parsefloat 认识小数
  • 隐私转换 定义变量 + 上一个0是不能成功转换的 -*/就可以成功转换

转义符

  • 把有特殊含义的字符转换回本意
  • 把普通字符转换出特殊含义

数据类型

  • 基本数据类型
  • typeof name 可查看当前变量的数据类型
  • 使用typeof得到的类型名是一个字符串

  • number数值型 string字符串型 boolean布尔型 undefined 未定义 null空

  • 复杂数据类型

  • object对象

短路运算两句话

  • 且运算是找到一个假就是假了 两个都对才是对
  • 或运算值找到一个真就是真了 两个都假才是假

零碎知识点

  • new Date().getDay()获取系统时间0是周日1-6对应周一到周六

  • switch(a)小括号的内容他比的是全等全等之后又break才会跳出不然就到最后一个default然后输出

  • 通过观察我们发现，尽管我们对基本类型值进行了类型转换，

  • 原始的数据是不会更改的，我们只是通过转换得到了一个新值。
  • +-*/%
    也可以将其他类型转换为数值型
  • -*/%
    都可以将其他类型转换为数值型

  • -0 *1 /1 %要比本身的值大就行（但是很麻烦，我们一般不用）

  • 在使用+操作转换的时候，将加号放在变量的最前面，就可以了

  • !!变量 隐式转换 可以将其他类型转换为bool类型（比较常用）

  • 一元，参与运算的操作数只有一个
